🚗 WashWave — Premium Car Care Discovery Platform

WashWave is a sleek, modern web experience designed to help users discover premium car wash and detailing garages with ease.
The platform focuses on luxury presentation, smooth user experience, and a mobile-first interface tailored for modern automotive services.

✨ Features


Premium dark-mode UI


Featured garage showcase


Garage detail experience


Service highlights


Instant booking via WhatsApp


Responsive modern layout


Smooth and clean user flow



🧭 User Flow
Homepage   ↓Featured Garages   ↓View Details   ↓Garage Details Page   ↓Book via WhatsApp

🖥️ Tech Stack


HTML5


CSS3


JavaScript (Basic DOM Manipulation)
